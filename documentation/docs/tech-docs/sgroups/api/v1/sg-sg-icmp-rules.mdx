# POST /v1/sg-sg-icmp/rules

## **Запрос**

`POST /v1/sg-sg-icmp/rules`

<ul>
  <li className="text-justify">
    если в теле запроса указано одно или более значений в обоих массивах sgFrom -\> sgTo, то получим ответ всех
    существующих комбинаций правил SG-SG:ICMP каждого указанного значения sgFrom с каждым указанным значением sgTo
    (value-to-value)
  </li>
  <li className="text-justify">
    если в теле запроса один из массивов пустой а во втором указаны от одного и более значений, то получим ответ всех
    существующих комбинаций правил SG-SG:ICMP каждого указанного значения со всеми существующими (any-to-value,
    value-to-any)
  </li>
  <li>
    если в теле запроса указаны пустые массивы sgFrom -\> sgTo, то получим ответ всех существующих комбинаций правил
    SG-SG:ICMP (any-to-any)
  </li>
  <li>
    если указано некорректное тело в запросе, то получим ответ всех существующих комбинаций правил SG-SG:ICMP
    (any-to-any)
  </li>
</ul>

```json
{
  "sgFrom": ["sg-0"],
  "sgTo": ["sg-1"]
}
```

## **Ответ**

```json
{
  "rules": [
    {
      "ICMP": {
        "IPv": "IPv6",
        "Types": [8]
      },
      "SgTo": "sg-3",
      "logs": false,
      "trace": false,
      "SgFrom": "sg-2"
    }
  ]
}
```
