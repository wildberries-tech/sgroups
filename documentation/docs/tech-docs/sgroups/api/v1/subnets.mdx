# GET /v1/sg/\{sgName\}/subnets

## **Запрос**

`GET /v1/sg/{sgName}/subnets`

## **Ответ**

```json
{
  "networks": [
    {
      "name": "nw-2",
      "network": {
        "CIDR": "10.150.0.222/32"
      }
    }
  ]
}
```

Код ошибки 404

- Указано неверное значение \{sgName\}

```json
{
  "code": 5,
  "details": [],
  "message": "SG 'string' is not found"
}
```

- Ошибка в запросе

```json
{
  "code": 5,
  "details": [],
  "message": "Not Found"
}
```
