# Security Groups

<div class="paragraph">
 Ресурс `Groups` представляет собой введенную нами абстракцию, которая позволят объединить подсети в логические группы и применять к 
 ним единые правила сетевого взаимодействия.
</div>

        #### Шаблон

        ```hcl
        chain FW-IN-sgName {
            # **********
            $Trace $Transport $TypeList $Counter $Log $Verdict    //ICMP
            # **********
            $Trace $Counter $Log $Verdict                         //Общие
        }

        chain FW-OUT-sgName {
            # **********
            $Trace $Transport $TypeList $Counter $Log $Verdict    //ICMP
            # **********
            $Trace $Counter $Log $Verdict                         //Общие
        }
        ```

        #### Пример использования

        ```hcl
        chain FW-IN-sg-example {
            # **********
            nftrace set 1 icmp   type { 0, 8 } counter packets 0 bytes 0 log level debug flags ip options accept  //ICMP
            nftrace set 1 icmpv6 type { 0, 8 } counter packets 0 bytes 0 log level debug flags ip options accept  //ICMP
            # **********
            nftrace set 1 counter packets 0 bytes 0 log level debug flags ip options accept                       //Общие
        }


        chain FW-OUT-sg-example {
            # **********
            nftrace set 1 icmp   type { 0, 8 } counter packets 0 bytes 0 log level debug flags ip options accept  //ICMP
        ```
