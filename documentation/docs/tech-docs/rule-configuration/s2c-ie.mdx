# Sgroup to CIDR (Ingress/Egress)

#### Описание

<div class="paragraph">
  Правило `IE:S2C` (Ingress/Egress Security Group to CIDR) предоставляет возможность гибкого управления и контроля за
  сетевым трафиком между различными группами безопасности и подсетями, используя TCP, UDP и ICMP протоколы. Это правило
  дает возможность детально настроить, какой трафик разрешен к передаче между группами безопасности и определенными
  подсетями, тем самым гарантируя высокий уровень защиты и управления сетевой инфраструктурой.
</div>
