# Sgroup to CIDR (tcp|udp)

### Описание

Правило в общей цепочке для входящего трафика от CIDR. Имеет третий приоритет в цепочке FW-IN-sgName.

### Шаблон

```hcl
chain FW-IN-sgName {
    # **********
		${Trace} ${RuleType} ${SrcCIDR} ${Transport} ${SrcPorts} ${DstPorts} ${Counter} ${Log} ${Verdict}
		# **********
}
```

### Пример использования

```hcl
chain FW-IN-sgname_example {
  # **********
  nftrace set 1 ip saddr { 1.1.1.1/8 } tcp dport { 80, 443 } counter packets 0 bytes 0 log level debug flags ip options accept
  # **********
}
```
