#### Шаблон

```hcl
chain FW-IN-sgName {
    # **********
    ${Trace} ${RuleType} ${SrcSgroup} ${Transport} ${TypeList} ${Counter} ${Log} ${Verdict}
    # **********
}
```

```hcl
chain FW-OUT-sgName {
    # **********
    ${Trace} ${RuleType} ${DstSgroup} ${Transport} ${TypeList} ${Counter} ${Log} ${Verdict}
    # **********
}
```

#### Пример использования

```hcl
chain FW-IN-sg-example-to {
    # **********
    nftrace set 1 ip saddr @familyName-v4-sg-example-from icmp type { echo-reply, echo-request } counter packets 0 bytes 0 log level debug flags ip options accept
    nftrace set 1 ip saddr @familyName-v6-sg-example-from icmp type { 0, 8 } counter packets 0 bytes 0 log level debug flags ip options accept
    # **********
}
```

```hcl
chain FW-OUT-sg-example-from {
    # **********
    nftrace set 1 ip daddr NetIPv4-sg-example-to icmp type { echo-reply, echo-request } counter packets 0 bytes 0 log level debug flags ip options accept
    nftrace set 1 ip daddr NetIPv6-sg-example-to icmp type { 0, 8 } counter packets 0 bytes 0 log level debug flags ip options accept
    # **********
}
```
